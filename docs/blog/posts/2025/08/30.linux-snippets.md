## get script dir

```
SCRIPT_DIR=$( cd -- "$( dirname -- "${BASH_SOURCE[0]}" )" &> /dev/null && pwd )
```

## create self signed

```
openssl req -x509 -nodes -subj "/C=CH/ST=ZH/O=IT/CN=example.org" -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365
```
